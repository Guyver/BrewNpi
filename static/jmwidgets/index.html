Welcome to JMWidgets

JMWidgets is an open source HTML5 widget library and HMI framework specifically designed for dashboards and automation applications. JMWidgets is a jQuery plug-in and uses the same interface as jQuery and jQuery UI widgets. To find out more about JMWidgets, visit us at www.jmwidgets.com.
